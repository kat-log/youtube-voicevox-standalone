# プライバシーポリシー

**YouTubeライブチャットずんだもんボイス読み上げ（APIキー不要モード搭載版）**

最終更新日: 2026年4月2日

---

## 収集する情報

本拡張機能は、個人を特定できる情報を収集・外部送信しません。

拡張機能が扱うデータは以下の通りです。

### ローカル保存のみ（外部送信なし）
- APIキー（YouTube Data API v3、VOICEVOX API）
- 話者設定・再生速度・音量などの設定値
- 読み上げ件数などの実績データ

これらはすべてブラウザのローカルストレージ（`chrome.storage.local`）にのみ保存され、開発者を含む第三者には送信されません。

---

## 外部サービスへの通信

拡張機能の機能に応じて、以下の外部サービスに通信します。

### DOMモード（APIキー不要）
YouTube ライブチャットページの DOM を直接読み取ります。外部 API への通信は発生しません。

### 公式APIモード
コメント取得のため、YouTubeが提供する **YouTube Data API v3** にリクエストを送信します。送信されるのは動画IDのみです。

### 音声読み上げエンジン
選択したエンジンに応じて通信が発生します。

| エンジン | 送信先 | 送信内容 |
|---------|--------|---------|
| WEB版VOICEVOX API (TTS Quest) | https://tts.quest/ | 読み上げるコメントのテキスト |
| ローカルVOICEVOX | http://localhost/ | 読み上げるコメントのテキスト |
| ブラウザ内蔵音声 | なし（ローカル処理） | — |

TTS Quest のプライバシーポリシーについては [TTS Quest 公式サイト](https://tts.quest/) をご確認ください。

---

## 第三者への情報提供

収集したデータを第三者に販売・提供することは一切ありません。
