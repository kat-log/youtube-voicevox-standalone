特徴

シンプルに、
でも本格的に。

3つのエンジンと豊富な設定で、あなたのYouTubeライブ視聴をより楽しく。

🎙️

APIキー不要で即使える

DOMモードで設定ゼロ。YouTubeライブを開いて「読み上げ開始」を押すだけ。 インストール直後からすぐ使えます。

🔊

3つの読み上げエンジン

ローカルVOICEVOX・WEB版API・ブラウザ内蔵音声から用途に合わせて選択。 本格的な音声合成から手軽な利用まで対応。

⚙️

豊富なカスタマイズ

話者・速度・音量・フィルター・並列再生まで細かく設定可能。 ランクアップする実績システムも搭載。

機能詳細

かゆいところに
手が届く機能。

  • 💻
    ローカルVOICEVOX（推奨） PCにVOICEVOXアプリをインストールして使用。回数無制限・最大5並列リクエスト対応。最高品質の音声。
  • ☁️
    WEB版VOICEVOX API クラウドAPIを利用。インストール不要でずんだもんをはじめ多キャラクター対応。APIキーで高速化可能。
  • 🌐
    ブラウザ内蔵音声 追加設定ゼロ。Chromeに搭載されたTTSエンジンを使用。手軽に始めたい方に。
  • 🎲
    ランダム話者モード コメントごとに異なる声で読み上げ。専用ページで話者を検索・選択。持ち回り制も対応（2〜10人）。
  • ⏩
    ラッシュモード（自動倍速） コメントが溜まったとき自動で読み上げ速度を上げて追いつく。しきい値も調整可能。
  • 🔀
    並列同時再生 最大10コメントを同時に読み上げ。ローカルVOICEVOX使用時に特に効果的。
  • 🚫
    NGワード / 文字数フィルター カンマ区切りのNGワードリスト（全角半角同一視）・デュアルスライダーで文字数範囲を設定。
  • ⌨️
    キーボードショートカット Alt+Shift+S で開始、Alt+Shift+Q で停止。マウスを触らず操作可能。
  • 🏆
    実績 & ランクシステム 読み上げ件数に応じて10段階でランクアップ。使い続けるほど称号が解放されます。

使い方

簡単3ステップで開始。

  1. 1
    Chrome Web Storeからインストール 下のボタンからChrome Web Storeページへ移動し、「Chromeに追加」をクリック。
  2. 2
    YouTubeライブを開く 読み上げたいYouTubeライブ配信のページ、またはアーカイブを開きます。
  3. 3
    「読み上げ開始」をクリック 拡張機能アイコンを押してポップアップを開き、「読み上げ開始」を押すだけ。APIキー不要で即スタート。
