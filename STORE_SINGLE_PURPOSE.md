YouTubeのライブ配信チャットコメントをVOICEVOXキャラクター（またはブラウザ内蔵音声）でリアルタイムに読み上げるChrome拡張機能です。

【コメント取得方式】
・DOMモード：APIキー不要！インストール後すぐに使用可能。ライブ開始前・配信中・アーカイブに対応
・公式APIモード：YouTube Data API v3を使用

【読み上げエンジン】
・ローカルVOICEVOX：PCにインストールしたVOICEVOXに直接接続。並列処理で高速化
・WEB版VOICEVOX API：ずんだもん・四国めたん等のキャラクターをAPIキーなしで利用可能
・ブラウザ内蔵音声：追加設定不要

【主な機能】
・持ち回り制・ランダム話者モードで複数キャラクターを使い分け
・並列同時再生・自動倍速ラッシュモードでコメントが多い配信にも対応
・NGワード・文字数フィルターによるコメント絞り込み
・コメント処理タイムラインでリアルタイムに状況確認
・キーボードショートカット（Alt+Shift+S：開始 / Alt+Shift+Q：停止）
・ダークモード・ログ表示・実績ランクシステム対応
・設定は即時反映＆自動保存
