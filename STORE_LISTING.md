📖 概要

YouTubeのライブ配信コメントをリアルタイムで音声に変換して読み上げる拡張機能です！

VOICEVOXの多彩なキャラクター（ずんだもん、四国めたん、春日部つむぎなど）、ブラウザ内蔵音声、またはローカルVOICEVOXエンジンで読み上げます！

【拡張機能で行っていること】
YouTubeコメントの取得
↓
VOICEVOX API、ブラウザ内蔵音声、またはローカルVOICEVOXエンジンで音声に変換
↓
音声の再生

✨ 機能紹介

【DOMモード（APIキー不要・おすすめ）】
・YouTube APIキー設定ゼロ！YouTubeページのコメントをそのまま読み上げ
・インストールしたらすぐ使い始められます
・ライブ配信ページを開いた状態で読み上げ開始するだけ！
・ライブ開始前でもOK！チャット欄が解放されていれば待機中から読み上げ開始できます

【3つの読み上げエンジン】
・ローカルVOICEVOXエンジン（推奨）
　→ PCにインストールしたVOICEVOXに直接接続！音声生成の回数制限なし
　→ 並列処理対応！同時リクエスト数（1〜5）を調整して音声生成を高速化
　→ 接続テスト機能付き。話者一覧も自動取得されます
・WEB版VOICEVOX API
　→ ずんだもん、四国めたん、春日部つむぎ ほか多数のキャラクターから選択可能！
　→ APIキーなしで動作。APIキーを設定すると生成が高速化されます
・ブラウザ内蔵音声
　→ 追加のAPIキー不要！PCにインストール済みの音声を利用

【音声調整】
・再生速度：0.1倍速〜3.0倍速（スライダー＋リセットボタン）
・音量：スライダーで細かく調整可能（※ノベルティ音声は音量固定）

【話者モード】
・ランダム話者モード：コメントごとにランダムで話者が切り替わる！専用ページで使う話者を自由に選択・検索可能
・持ち回り制話者モード：複数の話者を順番に割り当て。話者数（2〜10）を設定し、各スロットに好きな話者を指定可能
　→ 各スロットに「ランダム」も選べるので、固定とランダムの組み合わせも自由自在
※ランダム話者モードと持ち回り制話者モードは排他（どちらか一方のみ有効）
※3つのエンジンすべてで利用可能

【テスト再生（プレビュー再生）】
・ポップアップ内でお好みのテキストを入力してすぐに試聴可能
・全話者を一覧で試聴できる専用ページ付き！キャラクターごとにグループ表示、検索・一括操作にも対応

【並列同時再生】
・常時並列再生：常に複数コメントを同時に読み上げ（同時再生数を2〜10で調整可能）
・しきい値超過時のみ発動：待機コメント数がしきい値を超えた時だけ並列再生に切り替え（発動しきい値・同時再生数をスライダーで調整可能）
※ブラウザ内蔵音声では並列再生は利用できません

【キーボードショートカット】
・Alt+Shift+S（Mac: ⌥⇧S）：読み上げ開始
・Alt+Shift+Q（Mac: ⌥⇧Q）：読み上げ停止

【コメントフィルター】
・最新N件取得モード（取得件数を1〜10件で調整可能。コメント量が多い人気配信者で、古いコメントを読み続けてしまう時に！）
・文字数フィルター（最小・最大をデュアルスライダーで指定。短すぎる／長すぎるコメントをスキップ）
・絵文字のみのコメントをスキップ
・絵文字除去（コメント内の絵文字を取り除いて読み上げ）
・NGワードリスト（カンマ区切りで複数指定可能・全角半角同一視）
・NGワード動作モード切替（該当ワードのみ除去 or コメントごとスキップ）
・キュー自動キャッチアップ（しきい値超過時のみ発動：待機コメント数がしきい値を超えた時、自動で古いコメントを破棄して最新に追いつく。しきい値は調整可能）

【ラッシュモード（自動倍速モード）】
コメントが溜まってきたら自動で再生速度をアップ！キューが減ったら元の速度に戻ります。
開始しきい値・倍速時の速度・復帰しきい値をスライダーで細かく調整可能。
発動中はステータスバーに「AUTO」バッジが表示されるので、いつ切り替わったか一目でわかります！

【その他の便利機能】
・ダークモード（目に優しい表示に切り替え）🌙
・ステータス表示（接続状態・読上件数・キュー待機数をリアルタイム表示）
・ログ表示（レベル別表示（debug/info/warn/error）対応。アコーディオン表示＋専用ページで確認可能。ポップアップを閉じても保持）
・実績・ランクシステム（読み上げたコメント数に応じて10段階のランクアップ！ポップアップにランク表示＋専用の実績ページ）🏆
・VOICEVOX APIキー残高確認（残ポイントをワンクリックで確認可能）
・設定の即反映＆自動保存（すべての設定変更が即座に反映。保存ボタン不要で、毎回設定する必要なし！）
・エラー・リトライ情報表示（ステータスバーとログにエラー詳細やリトライ状況をリアルタイム表示）

🔧 使い方

【DOMモード（APIキー不要）】
１．拡張機能をインストールします
２．YouTubeのライブ配信ページを開きます
３．拡張機能アイコンをクリックしてポップアップを開きます
４．コメント取得方式が「DOMモード」になっていることを確認します
５．お好みの読み上げエンジン・話者・速度・音量を選びます
６．「読み上げ開始」ボタンをクリック！
７．楽しむ！🎉

【公式APIモード（YouTube APIキーあり）】
１．ポップアップのセットアップガイドから YouTube Data API v3 のAPIキーを取得・設定します
２．コメント取得方式を「公式APIモード」に切り替えます
３．あとはDOMモードと同じ手順で使えます

※DOMモードはライブ開始前・ライブ配信中・アーカイブ動画のコメントすべてに対応！
※公式APIモードはライブ配信中のみ機能します
※ショートカットキー（Alt+Shift+S / Alt+Shift+Q）でも操作できます

🔑 APIキーについて

【DOMモード】
APIキー不要！インストール後すぐに使えます。

【公式APIモード（YouTube Data API v3）】
ポップアップに表示されるセットアップガイドから手順を確認できます。
Google Cloud Consoleでプロジェクトを作成し、YouTube Data API v3を有効化してAPIキーを取得してください。
参考記事: https://qiita.com/MCK9595/items/fdbd543ff938febcd136#api%E3%82%AD%E3%83%BC%E3%81%AE%E6%9C%89%E5%8A%B9%E5%8C%96
（@MCK9595さんの記事）

【VOICEVOX API（任意・無料・設定で高速化）】
VOICEVOXエンジンはAPIキーなしでも動作しますが、キーを設定すると音声生成が高速化されます。
取得はこちらから: https://voicevox.su-shiki.com/su-shikiapis/
