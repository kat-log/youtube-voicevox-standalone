1. YouTube APIキーの取得と設定（必須）

コメントを取得するためには、無料のYouTube Data API v3のキーが必要です。 以下の手順で取得してください。

  1. Google Cloud Console にアクセスし、Googleアカウントでログインします。
  2. 上部から「プロジェクトの選択」を押し、「新しいプロジェクト」を作成します。
  3. YouTube Data API v3のページ を開き、「有効にする」をクリックします。
  4. 左側メニューの「認証情報」から「＋認証情報を作成」＞「APIキー」を選択します。
  5. 生成された文字列（APIキー）をコピーし、拡張機能の設定画面（YouTube API Key）に入力します。
※ APIキーは自分専用のものです。他人に教えたり、公開したりしないでください。

2. VOICEVOXとの連携（任意）

標準のブラウザ音声以外に、高品質な「VOICEVOX」の音声を利用できます。
ローカルVOICEVOXの導入を推奨しますが、WEB版APIも利用可能です。

2a. ローカルVOICEVOXの導入（推奨）

お使いのPCにVOICEVOXをインストールすることで、レート制限なし・高速で利用できます。

  1. VOICEVOX公式サイト からアプリをダウンロード・インストールします。
  2. VOICEVOXアプリを起動します（音声エンジンがバックグラウンドで自動的に起動します）。
  3. 拡張機能の「読み上げエンジン」で「ローカルVOICEVOX」を選択します。
  4. 「接続テスト」ボタンで接続を確認し、お好みの話者を選択します。
※ 読み上げ中はVOICEVOXアプリを起動したままにしてください。

2b. WEB版VOICEVOX API（代替）

ローカルインストールが難しい場合は、WEB版APIも利用できます。
APIキーの有無で2つのモードがあります。いずれも無料ですが、レート制限があり 連続使用時に一時的に音声生成が停止する場合があります。

WEB版VOICEVOX API（高速）— APIキーあり

APIキーを設定すると、WEB版の中ではより高速に音声を生成できます。

  1. su-shiki API 様のWebサイトへアクセスし、APIキーを発行・取得します。
  2. 拡張機能の設定画面の「詳細設定（VOICEVOX API）」を開き、取得したAPIキーを入力します。

WEB版VOICEVOX API（低速）— APIキー不要

APIキーなしでもそのまま利用できます。高速版と比べて応答が遅くなります。

  1. TTS Quest API の詳細はこちらをご確認ください。
  2. 特別な設定は不要です。読み上げエンジンで「WEB版VOICEVOX API」を選択するだけで利用できます。

3. 基本的な使い方

  1. YouTube Liveの配信ページを開きます。
  2. 拡張機能のアイコンをクリックしてパネルを開きます。
  3. 「読み上げ開始」ボタンを押すか、ショートカット（Alt+Shift+S / ⌥⇧S）を使用します。
  4. コメントの取得と読み上げが開始されます！停止したい時は「停止」ボタンを押してください。

※ 読み上げ設定（速度・音量・話者など）やフィルター機能は、配信中・停止中いつでも変更可能です。
