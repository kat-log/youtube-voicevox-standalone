1. YouTube APIキーの取得と設定（必須）

コメントを取得するためには、無料のYouTube Data API v3のキーが必要です。 以下の手順で取得してください。

  1. Google Cloud Console にアクセスし、Googleアカウントでログインします。
  2. 上部から「プロジェクトの選択」を押し、「新しいプロジェクト」を作成します。
  3. YouTube Data API v3のページ を開き、「有効にする」をクリックします。
  4. 左側メニューの「認証情報」から「＋認証情報を作成」＞「APIキー」を選択します。
  5. 生成された文字列（APIキー）をコピーし、拡張機能の設定画面（YouTube API Key）に入力します。
※ APIキーは自分専用のものです。他人に教えたり、公開したりしないでください。

2. VOICEVOXとの連携（推奨・任意）

標準のブラウザ音声以外に、高品質な「VOICEVOX」の音声を利用できます。
音声生成を高速かつ安定させるため、無料のWEB版VOICEVOX API（高速）APIキーの取得をおすすめしています。

  1. su-shiki API 様のWebサイトへアクセスします。（当拡張機能はVOICEVOXの公式提供ではなく、サードパーティのAPIを利用しています）
  2. 利用規約を確認後、ご自身用のAPIキーを発行・取得します。
  3. 拡張機能の設定画面の「詳細設定（VOICEVOX API）」を開き、取得したAPIキーを入力します。

3. 基本的な使い方

  1. YouTube Liveの配信ページを開きます。
  2. 拡張機能のアイコンをクリックしてパネルを開きます。
  3. 「読み上げ開始」ボタンを押すか、ショートカット（Alt+Shift+S / ⌥⇧S）を使用します。
  4. コメントの取得と読み上げが開始されます！停止したい時は「停止」ボタンを押してください。

※ 読み上げ設定（速度・音量・話者など）やフィルター機能は、配信中・停止中いつでも変更可能です。
